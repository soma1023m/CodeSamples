# Steps
* 1) Create a file hello.js.

* 2) Write and save the following code into the file.
```
var hello = function(){  
    print("Hello Nashorn");  
};  
hello(); 
``` 
* 3) Open terminal

* 4) Write command jjs hello.js and press enter.

After executing command, you will see the below output.

Output:
```
Hello Nashorn
```
